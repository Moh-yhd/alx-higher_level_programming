Using matrix_mul
----------------

First import "matrix_mul" from the "100-matrix_mul" module:
	>>> matrix_mul = __import__("100-matrix_mul").matrix_mul

		Now use it
		----------

Passing two (2X2) matrices:
	>>> matrix_mul([[1, 2], [3, 4]], [[1, 2], [3, 4]])
	[[7, 10], [15, 22]]

Passing two (2X2) matrices:
	>>> matrix_mul([[-1, 2], [3, -4]], [[1, -2], [-3, 4]])
